九、结语

Redis 与 Redisson 本身没有问题，根因在于系统层的字符编码不一致。

经验教训：

  • 容器 locale 是 JVM 编码的隐性依赖；
  • 生产环境应强制统一 file.encoding=UTF-8；
  • 加载脚本或配置文件时，应显式指定编码；
  • 避免在 Lua 等脚本中使用中文注释，以降低环境依赖。
